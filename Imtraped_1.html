Valid Palindrome - Leetcode

This one is to see if a phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward.

This was done in Python

        
            import re
            class Solution:
                def isPalindrome(self, s: str) -> bool:
                    palindrome = s.lower()
                    palindrome2 = re.findall("[a-z|0-9]", palindrome)
                    palindrome3 = palindrome2[::-1]
                    if(palindrome2 == palindrome3):
                        return(True)
                    else:
                        return(False)

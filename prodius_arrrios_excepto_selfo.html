Product of Array Except Self

Return the array of the products of the other elements except for itself

Solution:

        
            class Solution(object):
                def productExceptSelf(self, nums):
                    """
                    :type nums: List[int]
                    :rtype: List[int]
                    """
                    arr = [1] * (len(nums))
                    
                    prefix = 1
                    for i in range(len(nums)):
                        arr[i] = prefix
                        prefix *= nums[i]
                    postfix = 1
                    
                    for i in range(len(nums) -1, -1, -1):
                        arr[i] *= postfix
                        postfix *= nums[i]
                        print(nums[i])
                    
                    return arr

Majority Element - LeetCode

Given an array nums of size n, return the majority element.

Done in Python


        class Solution:
          def majorityElement(self, nums: List[int]) -> int:
            nums.sort()
            counts = [0] * (nums[-1] + 1)
            for i in nums:
              counts[i] += 1
            max = 0
            maxI = 0
            for i in range(len(counts)):
              if counts[i] > max:
                  max = counts[i]
                  maxI = i
            return(maxI)
